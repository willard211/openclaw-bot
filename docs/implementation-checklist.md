# 实施清单（V1）

## 一周内可完成
1. 准备三份知识库各 20 条高频内容（英文优先）。
2. 建立三条技能链路：产品、公司、销售话术。
3. 开启语言自动识别（en/zh/es 优先）。
4. 接入一个渠道（官网聊天或 WhatsApp 二选一）。
5. 配置人工接管：低置信度与高风险自动转人工。

## 验证标准
- 首响时间 < 10 秒
- FAQ 命中准确率 > 85%
- 转人工率（前两周）40%-60%（正常）
- 高风险回复 100% 审批

## 常见失败点
- 三个知识域混写，导致事实和话术污染。
- 销售话术缺少 CTA，回复不推进。
- 没有审批阈值，模型直接承诺价格或账期。
